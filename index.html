Speak My Chat

Speech with nothing to install

Make chat commands like !say for your users to get their message read out in the chat or get all of your chat get read out (Also useful for following chat on a channel if you're doing the washing up).
Different voices, cooldowns, blocked users, limit to mods, etc - of course!Try it here.

New Chatter List

it's super attentive

The first time one of your viewers chats their name will be added to a list, letting you know who's around without having to scroll through the chat history.
This tool will let you easily spot a chatter that might get buried under chat between your regulars or who's joined since you did a BRB.Try it here.

Chat Alerts

Don't miss out

If you don't check on the chat because you're caught up in your game or you're a smaller channel not used to getting many messages then this tool can alert you by flashing the browser or playing a sound (your choice). Cooldowns mean you won't get alerts for every message and you can set timeouts any time at the click of a button.Try it here.

You're muted!

Ding ding!

Alerts you when people say you're muted by flashing the browser or playing a sound.
You can limit it to mods, vips or named chatters and cooldowns will automatically trigger, or be extended with the click of a button.Try it here.

Known Chatters

good and bad

VIPs, named people or 'wrong uns' enter the chat - you get told
I'm considering a community driven wrong 'un list. You'd be notified by the tool as soon as they joined the channel even before they sent any messages with tags on why they'd been reported.

OBS Control via chat

Mod helpers

Custom chat commands will let mods, vips or named chatters change scenes (you left it on BRB again!) or even unmute your mic.
A little more complicated than the other tools as you'll need to install OBS Websockets - which you might already be using if if you're using tools like Touch Portal or Lioranboard.
!gamescene !brb !unmute, etc... you get the idea.

OBS Remote Control Panel

Mod like a boss

Easily set up a remote control panel that'll allow a mod to select scenes, mute and unmute your mic, etc. As with the chat commands if you've left yourself muted and on the BRB screen then mods can save the day.
After you've created a panel you can send a link to the panel page that'll be valid only for 6 hours, or you can enable panels for longer and revoke its use as any time.

A Bonus

That's nice

Most tools this can be added as a custom dock in OBS so they'll always be ready.
If you can't see OBS while gaming then chat commands like !muted cooldown 5 or !newchatter disable will let you or your mods stop things from getting really annoying.

Security

who's listening?

All of these tools bar one (mod remote panel, or if the community blacklist happens) will work directly from whatever browser you choose running entirely locally in javascript with zero communication to this site. This means you can download all or any of the tools and use them locally. You won't even need a local web server. You can edit the config files to add the channel(s) to load by default and set preferences on who can use use custom commands.
All tools can be limited to just your use as broadcaster or extend them with checkboxes to mods, vips or everyone. Use can even be limited to those named only where appropriate.
