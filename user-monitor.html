The users in channels pane takes a while to spot lurkers, so shurrup and wait.
There are wildcards
  • * match anything or nothing: spaces, weird characters, everything
  • ! optional space so you!is!smel matches you is smell, youissmell, youis smel, etc
  • ? optional non-space characters only so fr?gface will match frgface, frorrgface but not fro gface
  • + like ? but there must be at least one character so +git will match yougit but not git
  • ^ 10 chars of anything. Reduces really far apart matches that * might cause. tea^pot will match teapot, tea pot, tea pissy pot but not tea is really lovely pot. Add two for allowing 20 chars between, etc
OBS dock? Do all your settings, next door click start on miniview, copy the url for the dock = less gubbins showing up.
